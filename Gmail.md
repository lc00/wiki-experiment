## Inbox

* `j`/`k` - Move between messages
* `,` - Go to command dropdown
* `.` - Go to actions dropdown
* `u` - Back to threads
* ` ` ` -  Next tab
* `~` -  Previous tab

### Selection

* `x` - Select
* `* a` - Select all
* `* n` - Select none
* `* r` - Select read
* `* u` - Select unread

### Actions

* `e` - Archive
* `r` - Reply
* `a` - Reply All
* `Shift + t` - Add to tasks
* `c` - Compose
* `Ctrl + Enter` - Send
