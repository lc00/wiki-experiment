## Loop Structure

```bash
[Loop Header]
do
    # Loop Body
done
```

## Loop Headers

```bash
for i in $(ls -l)
```

```bash
for ((i=0; i<5; i++))
```

```bash
While
```

```bash
Until
```

## Loop Control

* `continue 3` - How many loops you want to jump up
* `break 3` - How many loops you want to jump up
* `wait [process_id]`
* `exit`
* `sleep 10` - (seconds)
