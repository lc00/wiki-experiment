## Arrow Functions
Instead of:

```js
function(event){
    //
}
```

Use this to pass the `this` context in:

```js
event => {
    //
}
```

shorthand function calls: `foo = bar.map(baz => baz + 1)`

## Assignment

## Classes

```js
class Shape {
    constructor (id, x, y){
        this.id = id;
        this.move(x, y);
    }
    move(x, y){
        this.x = x;
        this.y = y;
    }
    set width(width){
        this._width = width;
    }
    get width(){
        return this._width;
    }
}
```

## `for of`

```js
for (let n of fibonacci) {
    if (n > 1000)
        break;
    console.log(n);
}
```

## Generators

```js
function* range (start, end, step) {
    while (start < end) {
        yield start;
        start += step;
    }
}
```

## String functions

```js
[ 1, 3, 4, 2 ].find(x => x > 3); // 4
"hello".startsWith("ello", 1); // true
"hello".endsWith("hell", 4);  // true
"hello".includes("ell");      // true
```
